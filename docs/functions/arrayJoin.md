# $arrayJoin
Joins an array.
## Usage
> `$arrayJoin[name;separator?]`
## Parameters
|   Name    |                 Description                 |  Type  | Default value |
|-----------|---------------------------------------------|--------|---------------|
| Name      | The name for the array.                     | String | none          |
| Separator | The element separator to join the elements. | String | ,             |
