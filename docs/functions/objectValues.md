# $objectValues
Joins all valus from an object.
## Usage
> `$objectValues[name;separator?]`
## Parameters
|   Name    |                Description                |  Type  | Default value |
|-----------|-------------------------------------------|--------|---------------|
| Name      | The name for this object.                 | String | none          |
| Separator | The value separator to join the elements. | String | ,             |
