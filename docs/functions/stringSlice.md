# $stringSlice
Slices a text.
## Usage
> `$stringSlice[text;from;to?]`
## Parameters
| Name |           Description            |  Type  | Default value |
|------|----------------------------------|--------|---------------|
| Text | The text to work with.           | String | none          |
| From | Index to start slicing the text. | Number | none          |
| To   | End index to slice the text.     | Number | none          |
