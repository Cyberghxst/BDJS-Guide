# $stringIncludes
Check whether a string includes some of the provided words.
## Usage
> `$stringIncludes[text;values]`
## Parameters
|  Name  |      Description       |  Type  | Default value |
|--------|------------------------|--------|---------------|
| Text   | The text to work with. | String | none          |
| Values | Words to be checked.   | String | none          |
